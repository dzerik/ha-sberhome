SberHome — Авторизация

  1. Нажмите кнопку ниже — откроется страница входа через Сбер ID
  2. Войдите в свой аккаунт Сбера
  3. После входа браузер покажет ошибку — это нормально. Скопируйте URL из адресной строки (он начинается с companionapp://)
  4. Вернитесь сюда и вставьте URL в поле ниже
Войти через Сбер ID
Авторизация прошла успешно! Это окно можно закрыть.
Если URL не виден в адресной строке, откройте консоль разработчика (F12) и найдите ссылку companionapp://... в ошибке.
